## Welcome to xBlockOrigin

xBlockOrigin is a powerful browser extension that automatically mutes X.com (Twitter) users from specified countries using X's native country detection feature. Take control of your timeline with intelligent, automated filtering that works seamlessly across Chrome, Edge, and Firefox.

## How it works

When you browse X.com, xBlockOrigin works intelligently in the background:

<Steps>
  <Step title="Scan pages">
    The extension scans pages for user profiles across timeline, search, profiles, post detail pages, and notifications
  </Step>
  <Step title="Fetch user data">
    Fetches user ID and following status from X.com's UserByScreenName API in a single optimized call
  </Step>
  <Step title="Check whitelist">
    Verifies if the user is whitelisted and skips processing if they are
  </Step>
  <Step title="Check following status">
    If "mute following" is disabled, skips users you follow to protect your connections
  </Step>
  <Step title="Query country">
    Queries X.com's AboutAccountQuery API to get the user's originating country (cached for 24 hours)
  </Step>
  <Step title="Check blacklist">
    Compares the user's country against your configured blacklist
  </Step>
  <Step title="Hide posts instantly">
    Immediately overlays matching posts with a 50% opacity background and 8px blur, while preserving all post functionality
  </Step>
  <Step title="Mute via API">
    Automatically mutes the user via X.com's API if this is the first time encountering them
  </Step>
  <Step title="Save to database">
    Records the muted user to a local database for tracking and export
  </Step>
</Steps>

<Note>
The overlay approach preserves all post functionality - click handlers remain intact so you can interact with the post after unhiding it if needed.
</Note>

## Privacy first

xBlockOrigin respects your privacy:

- All data is stored locally on your device
- No data is sent to external servers
- Only communicates with X.com's official API
- Uses Chrome's secure storage APIs for data persistence
